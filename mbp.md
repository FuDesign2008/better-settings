# Mac 优化设置

## 说明

-   以下设置优化基于 MBP-15

## 优化原则

-   关闭不需要的功能(Less is More)
-   自动化
-   保证隐私

## 系统优化

### 系统更新

-   更新 Mac OS 系统至最新
    -    -> `About This Mac` -> `Software Update`

###  -> `System Preferences`

-   `General`

    -   取消 `Allow Handoff between this Mac and your iCloud devices`

-   `Dock`
    -   取消 `Magnification`
    -   Minimize windows using 选择 `Scale effect`
    -   取消 `Animate opening applications`
